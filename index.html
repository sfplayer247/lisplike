About

This language was created as an exercise. It uses lisp like syntax to make parsing easy.

Running code

To run code first you need import env.js and create an enviroment:

import Enviroment from "./env.js";
var env = new Enviroment();

An enviroment keeps its state when running multiple programs, because of this you will need to create a new enviroment to reset it.

Now to run the code, we simply call the run function with our code as a parameter:

env.run("(print (+ i1 i2))")

By itself the run function only returns the enviroment object, but the output can be accesed using Enviroment.output like so:

console.log(env.output) // Prints 3 to the console

We can view the values of all of the symbols (variables and functions) by looking at the symbol look up table:

console.log(env.symLUT)

Types

Number

Simple number type, needs no definition. Supports decimals and integers.

(print 2)

Symbol

Used to reference variables and functions. You can set the value of a symbol with the following:

(set x 100)

And you can retrieve the value just by typing the symbol:

(print x)

Boolean

Simple boolean, use lowercase true and false

Arrays

Simple array type, create an array using the arr function and retreive values with garr.

Property Lists/Objects

Property lists are the equivalent of objects or dictonaries. You can create a property list with the following:

(plist (arr 'a' 2) (arr 'b', 10))

And you can get properties from the object using a colon:

(print object:property)

Examples

Hello World:

(print 'Hello World')

Simple Math:

(void
 (print '2 times 2 equals: ' (* 2 2))
 (print '2 plus 2 equals: ' (+ 2 2))
 (print '2 minus 1 equals: ' (- 2 1))
 (print '8 divided by 2 equals: ' (/ 8 2))
 (print '2 to the power of 4 equals: ' (** 2 4))
)

Counting to ten:

(void
 (set x 0) ;set x to zero;
 (while (< x 10)
  (void
   (+= x 1) ;increment x by one every iteration;
   (print x)
  )
 )
)

Web Request

(void
 ask the user for a user to lookup;
 (set id (prompt 'user id:'))
 Make sure the ID is within bounds;
 (if (and (<= (toNum id) 10) (>= (toNum id) 0))
  (void
  Request the user data;
  (set user (decode (reqget (+ 'https://jsonplaceholder.typicode.com/users/' id))))

  Print out the user data;
   (print (+ 'ID: ' user:id))
   (print (+ 'Name: ' user:name))
   (print (+ 'Phone: ' user:phone))
   (print (+ 'Website: ' user:website))
   (print (+ 'Street: ' user:address:street))
   (print (+ 'City: ' user:address:city))
   (print (+ 'Zipcode: ' user:address:zipcode))
   (print (+ 'Suite: ' user:address:suite))
  )
  else statement;
  (
   print 'Invalid User ID'
  )
 )
)
